# analysis/

Populated in Phase 2–3 (see [METHODOLOGY.md](../METHODOLOGY.md)):

- `ratings/<name>.md` — blind human L0–L4 rating, written before the
  scanner's report is read.
- `critiques/<name>.md` — per-repository check-by-check critique: where the
  automated score and the human rating disagree, and why.
- `external-validity.md` — corpus-wide synthesis of Q1 (does the tool agree
  with expert judgment).
- `findings.md` — corpus-wide synthesis of Q2 (what the corpus reveals about
  gaps in the maturity model itself).
